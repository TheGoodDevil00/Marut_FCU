# Code of Conduct

Marut FCU is a student-originated open-source flight-control project. Participation in this repository should remain respectful, technically constructive, and safety-conscious.

## Our Standards

Contributors are expected to:

- communicate professionally about code, hardware, architecture, and test evidence
- welcome collaborators from software, avionics, ground-control, structures, and manufacturing disciplines
- accept technical critique without turning it personal
- prioritize correctness and flight safety above speed

Unacceptable behavior includes:

- personal attacks or dismissive conduct
- harassment in public or private project spaces
- misrepresenting unsafe or untested changes as validated
- sharing private information without consent

## Scope

This code of conduct applies across repository discussions, issues, pull requests, and project communication channels.

## Enforcement

Project maintainers may remove comments, close discussions, or revoke contribution access in response to violations. Before publishing the repository broadly, replace placeholder maintainer contact details in the support and security documents with a real monitored address.
